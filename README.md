jcasbin-vertx-plugin is an authorization middleware based on [jCasbin](https://casbin/jcasbin) for [Vert.x](https://vertx.io/)

## Installation

For Maven:

```xml
<dependency>
  <groupId>org.casbin</groupId>
  <artifactId>vertx-authz</artifactId>
  <version>${version}</version>
</dependency>
```
